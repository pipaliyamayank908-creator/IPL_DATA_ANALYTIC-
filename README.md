# IPL Data Analytics

This project analyzes IPL match data using Python, Pandas, Matplotlib, and Scikit-learn.

## Project Structure
- `dataset/` - raw IPL dataset
- `notebooks/` - Jupyter notebook version
- `IPL_CODE.py` - main Python analysis script
 

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## How to Run
```bash
pip install -r requirements.txt
python IPL_CODE.py
